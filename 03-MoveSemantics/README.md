# Move Semantics

This exercise is as an investigation of move semantics especially in C++20 and later.

## Requirements

- C++ compiler supporting C++20
- [CMake](https://cmake.org) >= 3.5

## Building

- see [../README.md](../README.md)

## Running

Run the sample tool
```console
$ <build-dir>/03-MoveSemantics/move_semantics
```

# Templates

Don't ask what templates can do for you, but what you can do for them.

## Requirements

- C++ compiler supporting C++20
- [Make](https://www.gnu.org/software/make/)

## Building

- Run `make`
  ```console
  $ cd 08-Templates
  $ make
  ```

## Running

- Run the sample tool
  ```console
  $ 08-Templates/templates
  ```
